LENGUAJES DE LA PROGRAMACIÓN

Lenguajes

Los podemos ver como la forma cotidiana en que nos comunicamos con otras personas, utilizamos una serie de palabras que están compuestas por letras, las cuales tienen un significado, y cada vez que las unimos formamos frases y significados que utilizamos como canal de comunicación con otras personas. Este conjunto de palabras y letras son códigos que tienen un sentido y pueden variar en cada idioma.

De esta misma forma nos comunicamos con una computadora, a la que le enviamos un mensaje o instrucciones sobre un algoritmo que hemos formado y le vamos a indicar que debe realizar ciertas operaciones, acciones, actividades y que devuelva un resultado.

Este conjunto de códigos forman un sentido lógico para ella, los traduce a la cantidad de bits que necesita para operar y devolver un resultado. Así es como se crea un canal de comunicación con la computadora, y esto que utilizamos para dicha comunicación es un lenguaje de programación.

💻Hay diferentes lenguajes de programación, pero lo que varía entre ellos es su sintaxis.

Javascript

JavaScript es el lenguaje de programación encargado de dotar de mayor interactividad y dinamismo a las páginas web. Cuando JavaScript se ejecuta en el navegador, no necesita de un compilador. El navegador lee directamente el código, sin necesidad de terceros. Por tanto, se le reconoce como uno de los tres lenguajes nativos de la web junto a HTML (contenido y su estructura) y a CSS (diseño del contenido y su estructura).

Declaradores de variables en JS

Al decir modificar me refiero al valor que almacenan.

  • Var: variable que se puede modificar y re declarar.
  • Let: solo se declaran una vez, pero pueden ser modificadas.
  • Const: no se pueden modificar, son constantes.

PHP

Características de PHP:

  • Se puede introducir código de PHP dentro de HTML.
  • Permite comunicarse directamente con un servidor.
  • Utilizado en el Backend.
  • Se puede conectar directamente con la base de datos
  • Al final de cada línea de código termina con un punto y coma.
